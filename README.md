# websites

Simple church website for RGWC Isiolo.

## Run locally

1. Install dependencies with `npm install`.
2. Start the app with `npm start`.
3. Open `http://localhost:3000`.

## Deployment notes

- Set `PORT` if your host assigns one.
- Add the Safaricom environment variables from `.env.example` if you want STK Push to work.
- Place the church logo image next to `index.html` using the filename `church logo.jpg`.
